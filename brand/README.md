# AIRED — Brand Kit

The core marks for AIRED. Dark, premium, metallic — a posh chrome/brushed-steel
finish on matte black. Built from scratch as hand-drawn geometric vector paths,
so the wordmark is pixel-exact and font-independent (no system font is used for
"AIRED").

Everything here is generated by [`generate.mjs`](./generate.mjs). Re-run it any
time; the outputs are deterministic.

```bash
node brand/generate.mjs
```

## Assets (`dist/`)

| File | Size | Use |
| --- | --- | --- |
| `aired-profile-1080.{svg,png}` | 1080×1080 | Profile picture. Lockup held inside a ~660px center safe-zone so it survives a circle crop. |
| `aired-cover-1640x856.{svg,png}` | 1640×856 | Facebook page cover. Bottom-left is left clear for the avatar overlap. |
| `aired-icon-512.{svg,png}` | 512×512 | App / favicon. Reads down to ~48px. |

Each asset ships as an exact-size **PNG** and a standalone **SVG**.

## Palette (the only colors used)

| Token | Hex | Role |
| --- | --- | --- |
| Background | `#0a0a0a` | matte black |
| Signal red | `#e10600` | the Red Line (deep / shadow tone) |
| Red highlight | `#ff2d2d` | brighter red (with `#ff3a30` sheen → `#b00500` shadow) |
| Wordmark base | `#ededed` | brushed metal (`#f5f5f5` top → `#6a6a6a` lower third) |
| Bronze inlay | `#d9b87a` → `#7a5c2e` | sacred-geometry rings, hexagram, corner filigree |

Pure black is used only to deepen the corner vignette.

## Metallic treatment

- **Wordmark** — brushed metal: a vertical highlight→shadow gradient, a specular
  copy nudged up and a shadow copy nudged down for an embossed/raised read, and
  very fine horizontal brushed striations clipped inside the strokes.
- **The Red Line** — polished red lacquer / glowing filament: a vertical sheen
  core with a bright centre filament, glow built from **layered translucent
  strokes** (never an SVG blur on a flat line — that collapses the filter), and a
  luminous fade at both ends via a horizontal mask.
- **Accents** — thin bronze inlay with a faint highlight edge; shimmering, not
  flat. Restrained, with lots of negative space.

## How it's built

- The `AIRED` letterforms are constructed as filled geometric vector paths
  (monoline: constant stroke weight). No font, no `<text>` — the wordmark is
  identical on every machine.
- The only text rendered from a font is the cover tagline
  *"AI-ed and proud · credited not thanked"*, set in a locally-installed serif and
  filled with a silver-metal gradient. The standalone SVG falls back through a
  serif stack, so it renders anywhere.
- Rasterized with `@resvg/resvg-js` (the engine `next/og` bundles). No fonts or
  assets are fetched from the network.

Every run prints a verification pass: it confirms each PNG's exact dimensions and
samples pixels to confirm the core palette (background, red, metal) plus the
bronze/silver accents are present.
